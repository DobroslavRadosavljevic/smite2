# Conquest Counter Swaps

## Swap order rule

Default replacement order:
- Slot 6 -> Slot 5 -> Slot 4

Keep core penetration timing online before large pivots.

## Matchup matrix

| Enemy problem | Swap in | Swap out first |
|---|---|---|
| Heavy healing | Brawler's Beat Stick, Contagion | The Crusher |
| Double tank | Titan's Bane earlier, The Executioner (if AA weaving) | The Reaper |
| Crit hunter fed | Spectral Armor, Hide of the Nemean Lion | The Reaper |
| Hard-CC chain | Magi's Cloak, Spirit Robe, Purification Beads | The Reaper |
| Burst one-shot | Shifter's Shield, Prophetic Cloak, Aegis path | The Reaper |
